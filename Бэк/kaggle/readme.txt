File descriptions
train.json - the training set, contains ~8000 news with predefined sentimens. You should train your model based on this data.
"text" column - raw text of each news

"id" column - unique identifier of each news

"sentiment" column - sentiment of each news (positive, negative or neutral)

test.json - the test set, contains ~2000 news without sentiments. You should use your trained model in order to identify these sentiments.
"text" column - raw text of each news. The main aim is to identify sentiments for each news (positive, negative or neutral).  

"id" column - unique identifier of each news
